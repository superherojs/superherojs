Superhero JavaScript

Creating, testing and maintaining a large JavaScript code base is not easy — especially since great articles on how to do this are hard to find. This page is a collection of the best articles we've found on the topic. We recommend reading the articles in order. Have fun!

  1. Understanding JavaScript: Syntax, style and gotchas
  2. Organizing your code: API design, deferreds, files and folders
  3. Testing your application: Testable code, readable tests
  4. Tools of the trade: Workflow, developer tools, debugging tips
  5. Performance: Memory use, performance profiling, browser redraws
  6. Upcoming: New things coming to JavaScript
  7. Next steps: Books and other resources
